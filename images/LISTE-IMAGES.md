# 📸 Images du site Ramya — mode d'emploi

Toutes les images du site sont maintenant dans ce dossier `images/`.

## Comment changer une image
1. Prépare ta photo (format `.jpg`, idéalement largeur ~1200 px, poids < 300 Ko).
2. Renomme-la **exactement** comme le fichier à remplacer ci-dessous.
3. Place-la dans ce dossier `images/` (écrase l'ancienne).
4. Redéploie le dossier `site` sur Netlify.

> Astuce : tu peux changer juste les images que tu veux, laisser les autres.
